# Onboarding e Tutorial para Novos Usuários

Experiência de primeiro uso e descoberta das funcionalidades:

1. Tour guiado na primeira visita (tooltips ou overlay passo a passo)
2. Destacar War Room, Consultant, Opportunities e Journal
3. Opção "Não mostrar novamente" e persistir em preferências
4. Checklist inicial opcional (ex: "Crie sua primeira oportunidade")
5. Dicas contextuais em features avançadas (RAG, automação, templates)
6. Vídeo ou GIF curto opcional na página de boas-vindas
7. Link "Ajuda" ou "Como usar" acessível no header/settings
8. Modo "demonstração" com dados de exemplo para explorar sem comprometer dados reais

**Arquivos esperados:**
- `src/components/Onboarding/Tour.tsx` ou uso de lib (react-joyride, driver.js, etc.)
- `src/components/Onboarding/WelcomeModal.tsx` ou página de boas-vindas
- Preferência em user profile ou localStorage para tour concluído
- Seção de ajuda em Settings ou componente HelpPanel.tsx
